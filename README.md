# fzfind #
> fzf based file fuzzy finder for Bash

![](showoff.gif)

### Dependencies ###
+ GNU coreutils
+ fzf

### Instalation ###
1. Run `make install` (to compile the last word finder C program and move it into your $PATH)
2. Source `fzfind.rc` from your `.bashrc` or copy its contents directly into it
